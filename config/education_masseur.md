# 🖐 {{VERTICAL_NAME}} — Обучение для массажиста

## Быстрый старт

Хочешь попробовать запись сеанса? Вот полный цикл:

1. **Открой Mini App** → ⚙️ → **Массажист**
2. Вкладка **«Массажист»** → **«Клиенты»**
3. Нажми на Анну Тестову (создаётся админом через кнопку «🧪 Создать тестового»)
4. Увидишь **золотую предсеансовую сводку** — что беспокоит, что рекомендует AI
5. Нажми **«🎥 Начать сеанс массажа»** → выбери длительность → **«Начать»**
6. Сними видео (камера + микрофон) → **«Завершить»**
7. Видео придёт тебе в Telegram с подписью «🎥 Сеанс массажа»
8. Нажми **«📓 Добавить запись в дневник»** — заполни: техника, интенсивность, отзыв клиента

> **Клиент без анкеты?** Кнопка будет «🎬 Тренировка / тест» — видео уходит без записи в дневник.

## Ваша роль
Как массажист, вы получаете доступ к панели управления клиентами, предсеансовым сводкам и дневнику сеансов. Ваша задача — использовать рекомендации AI для максимально эффективной работы с каждым клиентом.

## Как открыть панель массажиста
1. Откройте Mini App
2. Нажмите ⚙️ (шестерёнка) в правом верхнем углу
3. Переключайтесь между ролями: **Клиент → Админ → Массажист**
4. В режиме массажиста появляется вкладка **«Массажист»**

> Режим сохраняется между сессиями.

## Вкладка массажиста
### 📋 Список клиентов
Все клиенты, прошедшие консультацию. По каждому видно:
- Количество консультаций
- Дату последнего визита
- Телефон

### 🧑‍🤝‍🧑 Карта клиента (нажмите на имя)
#### 📋 Предсеансовая сводка
Перед каждым сеансом открывайте карту клиента. Сводка содержит:
- **💬 Жалобы** — что беспокоит клиента
- **📍 Локация боли** — конкретная зона
- **🩹 Тип боли** — острая, хроническая, тянущая
- **🖐 AI рекомендует** — техника, которую подобрал искусственный интеллект
- **⚠️ Противопоказания** — важные ограничения (если есть)

> Всегда проверяйте сводку перед началом сеанса!

#### 🕐 История консультаций
Хронология всех визитов клиента: дата, рекомендованная техника, жалобы, музыка.

#### 🛤️ Путь клиента
Визуальный граф, показывающий **все визиты клиента и что изменилось между ними**:
- Каждый визит — узел на линии времени
- Между узлами — **дельта изменений**: локация боли, техника, противопоказания, музыка
- Красная точка = есть противопоказания
- Золотая точка = всё в порядке
- Дельта веса окрашена: зелёный (улучшение) / красный (ухудшение)

> Путь клиента — лучший способ увидеть прогресс за 1 секунду. Открывайте его перед каждым сеансом, чтобы понимать динамику.

#### 📓 Дневник сеансов
Ваши записи после каждого сеанса. Показывают динамику: что планировалось vs что выполнено.

### Заполнение дневника
После каждого сеанса нажмите **«📓 Добавить запись в дневник»**.

**Поля:**
| Поле | Описание |
|------|---------|
| 📋 Планировалось (AI) | Заполняется автоматически из рекомендации AI |
| 🖐 Выполнено | Что реально делали (может отличаться от плана) |

### 🎥 Запись сеанса
В карте клиента есть кнопка **«🎥 Начать сеанс массажа»**:

1. Выберите длительность: 30 / 60 / 90 мин
2. Нажмите **«▶ Начать»** — запускается таймер и видеозапись
3. Запись идёт с камеры телефона (или только аудио, если нет доступа к камере)
4. Можно поставить **⏸ Пауза** и **▶ Продолжить**
5. По окончании таймера — хаптик + уведомление
6. **⏹ Завершить** — видео отправляется в ваш личный чат с ботом
7. В дневник автоматически добавляется запись о проведённом сеансе

> Запись сеанса — отличный способ самоанализа и создания контента для канала. Нарезайте куски, делитесь техниками.
> 
> **Важно:** если у клиента нет анкеты здоровья — кнопка будет «🎬 Тренировка / тест», видео придёт без записи в дневник. Только для тренировки и тестирования системы.

### 🎥 Запись видео в чате со специалистом
Помимо записи сеанса из карты клиента, вы можете **снимать видео прямо в чате с ИИ-специалистом** (например, с Видеоаналитиком):

1. Откройте чат с любым специалистом (💬 Чат → выберите или создайте)
2. Нажмите **📹** (кнопка рядом с 📎)
3. Откроется камера — нажмите **▶ Начать**
4. Во время записи можно:
   - **⏸ Поставить на паузу** — если нужно прерваться
   - **📍 Отметить момент** — добавьте отметку с таймкодом и комментарием (например: «клиент напрягает плечи», «неудобное положение рук»)
5. **Сегментная запись (до 1 часа):**
   - Видео автоматически делится на сегменты по **5 минут**
   - Каждый сегмент приходит в ваш Telegram как отдельное видео
   - После завершения скачивается **файл для склейки** (`session_files.txt`) — список всех сегментов для ffmpeg concat
   - На устройстве: `ffmpeg -f concat -i session_files.txt -c copy session.mp4`
6. Нажмите **⏹ Завершить** — все сегменты отправлены, отметки переданы специалисту на анализ
7. ИИ-специалист проанализирует видео по кадрам и даст рекомендации

> **Почему сегменты:** Telegram лимит — 50 МБ на файл. 5 мин WebM ≈ 15-20 МБ. 1 час сеанса = 12 сегментов. Сервер не хранит видео — сегменты остаются в Telegram для скачивания.
> **Для кого:** Видеоаналитик — специальный агент для оценки техники массажа. Он анализирует: положение рук, темп, симметричность, реакцию клиента. Отличный инструмент для самообучения и повышения качества.

## 📓 Дневник массажиста

После сеанса (или отдельно) можно добавить запись в дневник. Поля формы:

| Поле | Описание |
|------|---------|
| 📋 Планировалось | Предзаполняется из AI-рекомендации |
| 🖐 Выполнено | Что сделали по факту |
| 💪 Интенсивность | Лёгкая / Средняя / Высокая |
| 🧴 Средства | Масла, кремы, инструменты |
| 🔬 Состояние тканей | Ваши наблюдения |
| 💬 Отзыв клиента | Что сказал клиент |
| 📌 Рекомендации | Что порекомендовали на дом |
| ⭐ Оценка | От 1 до 5 |
| 📝 Заметки | Любые дополнительные наблюдения |

> Поле **«Планировалось»** предзаполнено из AI-рекомендации. Если вы изменили технику — запишите актуальную в «Выполнено». Это помогает отслеживать эффективность AI.

## Что делать, если...
- **Нет сводки** — клиент прошёл консультацию? Если да, обновите страницу
- **Рекомендация не точна** — вы всегда можете применить свою экспертизу. Запишите в дневник, что сделали и почему
- **Клиент жалуется на новую боль** — откройте карту, проверьте противопоказания, отметьте в дневнике

## Как интерпретировать рекомендации AI
AI-рекомендация строится на:
1. Анкете здоровья клиента
2. Фото/видео анализе (если загружены)
3. Предыдущих сеансах

Рекомендация — **отправная точка**, а не строгий протокол. Ваш опыт важнее. Если AI рекомендует классический массаж, а вы видите, что нужен лимфодренаж — делайте лимфодренаж и зафиксируйте это в дневнике.

## 📅 Система записи (Booking)

Клиенты могут записываться на сеанс двумя способами:

### Как приходят уведомления
1. **Клиент создаёт запись** → вы получаете уведомление в Telegram с кнопками ✅ Подтвердить / ❌ Отменить:
   ```
   📅 Новая запись на сеанс
   👤 Клиент: Анна / ID 123456
   💆 Услуга: Классический массаж
   📆 Дата: 2026-06-15
   ⏱ Время: 10:00
   ```
2. **За 3 часа до сеанса** → если запись не подтверждена, придёт напоминание
3. **За 1 час до сеанса** → если запись не подтверждена, она **автоматически отменяется**. Вы получите уведомление: «❌ Автоотмена записи»
4. **Клиент подтверждает** → второе уведомление
5. **Клиент отменяет** → уведомление об отмене

### Что делать при новой записи
1. Откройте чат с ботом — уведомление пришло туда
2. Подтвердите запись (если нужно) или свяжитесь с клиентом
3. Запись автоматически отображается в разделе «Мои записи» у клиента

### Статусы записи
| Статус | Значение |
|--------|---------|
| ⏳ Ожидает | Клиент создал, но ещё не подтвердил |
| ✅ Подтверждён | Клиент подтвердил — сеанс состоится |
| ❌ Отменён | Запись отменена (клиентом или массажистом) |
