# ⚙️ AI Prophet — Обучение для администратора

## Панель администратора
Вы управляете салоном через Mini App. Для входа не нужен пароль — система определяет вас по Telegram ID.

### Как открыть админ-панель
1. Откройте Mini App
2. Нажмите ⚙️ (шестерёнка) в правом верхнем углу
3. Переключайтесь между ролями: **Клиент → Админ → Массажист**
4. В режиме админа появляется вкладка **«Админ»**

> Режим сохраняется между сессиями.

### 🧪 Тестовые пациенты
Для демонстрации и обучения можно создать тестовых пациентов:

- **«🧪 Создать тестового»** — генерирует пациента с реалистичными данными и историей консультаций
- Тестовые помечены бейджем `🧪 Тест` и выделены серым цветом
- Не влияют на реальную статистику
- Можно удалить кнопкой **«🗑 Удалить тестового»**

### 🎓 Обучение

## Список пользователей и статус анкеты

В админ-панели отображаются **все пользователи**, которые хоть раз взаимодействовали с ботом или Mini App.

- **✅ Есть анкета** — у клиента заполнена анкета здоровья, доступна предсеансовая сводка
- **⚠️ Нет анкеты** — клиент не проходил консультацию, кнопка записи сеанса доступна, но предсеансовой сводки нет

Пользователи без анкеты отображаются **внизу списка**. Чтобы заполнить анкету — отправьте клиенту ссылку на бота или откройте 🖐 Массаж → «Начать консультацию».

## Управление массажистами
Администратор может назначать массажистов:
- В карте клиента кнопка **«➕ Назначить массажиста»**
- Массажист получает доступ к панели и дневнику

## Сеанс vs Тренировка

В карте клиента кнопка записи видео различается в зависимости от статуса анкеты:

| Условие | Кнопка | Назначение |
|---------|--------|------------|
| ✅ Анкета есть | **🎥 Сеанс массажа** | Реальная работа с клиентом. Видео уходит массажисту в Telegram + запись в дневник |
| ⚠️ Анкеты нет | **🎬 Тренировка / тест** | Тестирование системы, тренировка приёмов, создание контента. Видео уходит массажисту **без** записи в дневник |

**Принцип:** без анкеты здоровья — никакой записи в дневник. Только тест системы.

## Команды Telegram для админа
| Команда | Действие |
|---------|---------|
| `/id` | Узнать свой Telegram ID |
| `/give_access @username` | Дать доступ администратору |
| `/revoke_access @username` | Забрать доступ |
| `/admin_ids` | Список всех администраторов |

> **Важно:** доступ можно давать только через `@username`, не через ID. Пользователь должен написать что-нибудь боту перед выдачей доступа.

## Команды владельца
Владелец (`OWNER_USERNAME`) может всё, что админ, плюс:
- `/vip` — управление VIP-режимом
- `/reset` — сброс пароля

## Советы
- **Создайте 2-3 тестовых пациента** для демонстрации салона новым массажистам
- **Проверяйте дневники** массажистов — видно, что планировалось vs выполнено
- **Используйте статистику** для понимания популярных техник и закупки расходников
- При добавлении нового массажиста выдайте ему доступ и покажите вкладку 🎓 Обучение

## 🎬 Обучающие видео для персонала
В файлы обучения можно встраивать видео двумя способами:
- `[YOUTUBE: video_id]` — встроенный YouTube-плеер (например, `[YOUTUBE: dQw4w9WgXcQ]`)
- `[VIDEO: https://...]` — прямая ссылка на MP4-файл

Как добавить новое видео:
1. Загрузите видео на YouTube (или хостинг с прямой ссылкой на MP4)
2. Откройте `config/education_masseur.md` или `config/education_admin.md`
3. Добавьте строку `[YOUTUBE: id_вашего_видео]` в нужное место
4. Сохраните, запушите — обучение обновится автоматически
