## 📋 1. КРАТКОЕ ОПИСАНИЕ ПРОЕКТА

## 🚀 6. РАЗВЁРТЫВАНИЕ

### Docker:

```dockerfile
FROM python:3.11-slim
RUN apt-get update && apt-get install -y ffmpeg dnsutils
WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
COPY . .
CMD ["python", "main.py"]
```

### CI/CD Pipeline:

1. `git push` → триггер GitHub Actions
2. Action пушит код на HF Spaces
3. HF автоматически пересобирает Docker-контейнер
4. Бот перезапускается с новым кодом

### Переменные окружения:

```env
TELEGRAM_TOKEN=ваш_токен
GEMINI_API_KEY=ваш_ключ_google
HF_TOKEN=ваш_токен_huggingface
VIP_PASSWORD=prophet2026
VIP_RESET_PASSWORD=reset2026
OWNER_USERNAME=dizel0110
PORT=7860
```
