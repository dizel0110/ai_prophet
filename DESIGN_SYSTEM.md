# 🎨 Design System — AI Prophet (Massage Salon Mini App)

> Для AI-агентов: вся палитра, анимации, UX-принципы и ограничения.

---

## 1. 🆔 Бренд-айдентика (ключевой инсайт)

Массажный салон **не про «модный дизайн»**, а про **тёплую, уютную, ламповую атмосферу**.

**Палитра** — не просто "красивые цвета", а айдентика:
- `#2c1810` (коричневый) — земля, дерево, основательность
- `#c49a6c` (золото) — тёплый свет, премиальность, расслабление
- `#a67c4e` (тёмное золото) — глубина, спокойствие
- `#f8f5f0` (кремовый) — натуральность, чистота, уют

**Золото ≠ акцент.** Это основа бренда. Массаж — про тепло рук, а золото — самый тёплый цвет.

Источник вдохновения: интерьеры массажных кабинетов, спа-салонов, свечи, дерево, натуральные ткани.

### Что НЕ подходит:
- Холодные тона (синий, фиолетовый, холодный серый)
- Неон, кислотные цвета
- Тёмная тема в отрыве от Telegram (см. п.3)

## 2. 🎬 Анимационная система

### Принципы:
1. **Только GPU-свойства** — `transform`, `opacity`. Никаких анимаций `width`, `height`, `margin`, `background` (кроме `::before`).
2. **CSS-only** — без JS-библиотек (anime.js, GSAP). Всё через `@keyframes`.
3. **Apple-like spring** — `cubic-bezier(.16,1,.3,1)` для всех entry-анимаций.
4. **Микро-анимации** — каждый тап подтверждается визуально (scale + тень).
5. **prefers-reduced-motion** — пользователь контролирует на уровне ОС.

### Tab Animations (8 вкладок):

| Вкладка | CSS-класс | Анимация | Смысл |
|---------|-----------|----------|-------|
| 🏠 Главная | `.tab.active[data-page="home"]` | `tabHome` — золотое свечение (filter + text-shadow) | Тепло очага |
| 💆 Услуги | `.tab.active[data-page="services"]` | `tabFloat` — парение translateY(-4px) | Лёгкость рук |
| 👫 О нас | `.tab.active[data-page="about"]` | `tabBreathe` — scale 1.1↔1.2 | Дыхание, жизнь |
| 📞 Запись | `.tab.active[data-page="booking"]` | `tabRipple` — быстрый scale 1.1→1.25→1.1 | Волна контакта |
| 🧑‍⚕️ AI | `.tab.active[data-page="ai"]` | `tabPulse` — opacity + filter | Мерцание нейросети |
| 💬 Чат | `.tab.active[data-page="chat"]` | `tabBlink` — opacity 1↔.35 | Моргание ожидания |
| 🎵 Музыка | `.tab.active[data-page="music"]` | `musicFloat` — translateY(-4px) | Ноты в воздухе |
| 📋 Ещё | `.tab.active[data-page="more"]` | `tabSlide` — translateX(4px) | Листание контента |

### Page Entry:
- Единая анимация: `pageEntry` — translateY(16px) + scale(.97) → identity
- Время: 0.45s, spring-ускорение
- Services page: дополнительный stagger на 12 `.svc` (каждый через 35ms)
- Home page: статистика (3 блока) pop-in каскадом (100ms, 200ms, 300ms)

### Hero:
- Существующий `::before` glow (радиальный градиент, вращение)
- 7 CSS-частиц (золотые точки, дрейф через custom properties)
- `overflow:hidden` — частицы не выходят за hero

### Card Micro-interactions:
- `:active` → `transform:scale(.98)` + `box-shadow` усиление
- `transition` на `transform` + `box-shadow` (0.25s, spring)
- `will-change: transform` НЕ ставить (лишняя память)

## 3. 🌗 Тема (День/Ночь)

**Не нужна. Это уже встроено в Telegram.**

```css
--bg: var(--tg-theme-bg-color, #f8f5f0);
```

Telegram сам прокидывает свою тему (светлую или тёмную) через CSS-переменные `tg-theme-*`. Если у пользователя Telegram в тёмной теме — Mini App автоматически подхватит тёмный фон.

Но наша **кастомная палитра** (золото + коричневый) — это бренд, и она должна оставаться доминирующей независимо от темы Telegram. Поэтому `--gold`, `--gold-dark`, `--brown` заданы явно, а `--bg` и `--text` наследуют тему Telegram.

**Не добавлять переключатель темы** — это нарушает:
- Принцип "кратчайший путь клиента" (лишняя кнопка)
- Единообразие бренда (массажный салон = тёплые тона)
- Бесшовность Telegram (Mini App не должен спорить с родительским приложением)

## 4. 📱 Производительность (HF Spaces constraints)

| Ресурс | Значение |
|--------|----------|
| CPU | 1 vCPU (очень слабый) |
| RAM | 8 GB (достаточно) |
| Storage | 50 GB (~45 GB свободно) |
| Network | Внешние API (HF Router, Gemini) |

### Правила:
- Все анимации — CSS GPU, 0 нагрузки на CPU
- Нет видео-файлов, GIF, тяжелых изображений
- Нет JS-библиотек анимаций
- Нет WebGL, Canvas 2D — только DOM + CSS
- Всё в одном HTML-файле (никаких дополнительных запросов)
- Иконки — emoji (0 трафика)
- Telegram WebApp JS — кешируется браузером Telegram

## 6. 📐 Разметка страниц

```
hero (общий для всех страниц)
├── h1 + divider + sub + badge
└── particles

8 страниц (.page):
├── #page-home    — статистика, популярные услуги, музыка, кнопка записи
├── #page-services — полный прайс (12 услуг)
├── #page-about   — описание, мастера, сертификаты, галерея
├── #page-booking — контакты, часы, форма записи
├── #page-ai      — AI-консультация, список специалистов
├── #page-chat    — чат со специалистами (список + поиск)
├── #page-music   — жанры → треки → плеер
└── #page-more    — отзывы, блог, плейлисты, канал

tab-bar (8 табов, fixed bottom)
modal-overlay (специалисты, формы, галерея)
player-bar (fixed bottom, поверх tab-bar, conditional)
```

## 7. 🎵 Music Equalizer (CSS)

```css
.player-eq span { animation: eqBounce .55s ease-in-out infinite alternate; }
/* 5 полос с разной высотой (35%-90%) и animation-delay */
```

Работает только при активном воспроизведении (класс `.active`). Гаснет на паузе.
